Git Basics

Initialize a repository:

git init

Clone repository:

git clone reponame

Pull changes from remote:

git pull

Add all files to staging:

git add .

Removes and stages the file removal:

git rm filenme

Inline commit:

git commit -m "message"

Push changes to remote:

git push

Revert the changes to head:

git reset --hard HEAD

git clean -f -d

Unstage all changes:

git reset HEAD -- .

Overwrite local changes with the state of the last commit:

git checkout -- file

Compare working directory with staging area (no output if files are staged):

git diff

Compare staged changes with the last commit:

git diff --staged

Store changes not ready to commit on the stack:

git stash

Apply and drop the changes from stack:

git stash pop

Viewing commit history in one line:

git log --oneline

List shortnames of each remote with urls:

git remote -v

Set an alias:

git config --global alias.unstage 'reset HEAD --'

Change branch:

git checkout branchname

Create a new branch and switch to it:

git checkout -b newbranch

Show unmerged files during a conflict:

git merge > git status

Generate SSH-key:

ssh-keygen -t ed25519 -C "email@example.com"

Git Tutorial:

man gittutorial
